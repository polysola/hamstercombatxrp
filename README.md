# Vortex Ton

Vortex Ton - The Next Generation GameFi on TON Blockchain.

## Overview

This project is a premium GameFi experience built on the TON Blockchain.

## Getting Started

To get started, follow these instructions:

### Installation

1. **Install Dependencies**:

    ```bash
    npm install
    ```

2. **Run the Development Server**:

    ```bash
    npm run dev
    ```

3. **Open Your Browser**:

    Navigate to localhost to see the application.

## Technologies

- React
- TypeScript
- Tailwind CSS
- Firebase
